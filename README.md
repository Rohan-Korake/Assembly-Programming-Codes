## 🚀 How to Clone and Run

1. Clone this repository

```bash
https://github.com/Rohan-Korake/Assembly-Programming-Codes.git
```

2. To Assemble

```bash
TASM Filename.ASM
```

3. To link

```bash
LINK Filename.OBJ
```

4. To Debug

```bash
TD Filename
```
